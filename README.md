# portfolio

Cassette-tape themed portfolio site — live Canvas gameplay previews, behavioral scroll tracking, unlock progression. Dark mode only.

This is an experimental portfolio surface. For the curated showcase used in current applications, see [Moranetz/selected-work](https://github.com/Moranetz/selected-work) — live at [moranetz.github.io/selected-work](https://moranetz.github.io/selected-work/).

## Stack

- Static HTML + CSS + JavaScript
- Canvas-rendered gameplay previews (no framework)
- Dark-mode only
